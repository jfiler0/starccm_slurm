The purpose of this repository is to provide a set of editable examples for using STAR-CCM+ with the Virginia Tech ARC systems so that more people at the Virginia Tech PHASE research group can make use of their resources. However, I also struggled to find public information about how to run STAR-CCM+ in slurm mode (especially for the design manager and with Java macros). So, I hope this repository can make someone's life simpler in the future.
The associated PDF provides detailed information about how to run each script with notes on the purpose of different lines. Note the following caveats:
1) Many of the scripts reference a specific file path. You will need to edit them to the appropriate username and directory or it will immediately fail.
2) There is a ton more complexity in terms of additional SLURM arguments, macros, and the design manager. The goal of this is only to provide a working base to edit and explore from.
3) I make no guarentees as to how these programs will behave in your hands or on your server.
This work was done while at Virginia Tech while I worked on this paper (shameless plug): https://arc-aiaa-org.ezproxy.lib.vt.edu/doi/epdf/10.2514/6.2026-0246
